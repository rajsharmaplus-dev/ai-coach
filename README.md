
# Sanai: Precision AI Interview Coach

Sanai is a premium, real-time AI interview coaching platform. It uses advanced multimodal AI to conduct realistic mock interviews, providing candidates with deep behavioral and technical analysis to help them land their dream roles.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
